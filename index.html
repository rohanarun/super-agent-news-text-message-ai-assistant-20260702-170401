How Super handles a texted request

1. You send a message

“Check my airline credit and remind me before it expires.”

2. Super plans the steps

The agent determines which sites, logins, and pages are required, similar to how a human would.

3. Computer use

Super opens a browser, signs in where appropriate, and completes the workflow with user confirmation.

4. Cache reuse

The flow is stored in the computer‑use cache, reducing friction and cost for future requests.
